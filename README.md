Here, I have craeted three set of programs, where I gradually developed each one basic to little complex.
1. Folder pokemon
   |-> has very basic code which contains only Main.java and Pokemon.java
   |-> In main.java, I have hard coded each pokemon type using some basic logic in if else loop to calculate its power and set its type.

2. Folder PokemonUpgraded
   |-> has main.java, pokemon.java and pokemoninfo.java files using arrays.
   |-> pokemoninfo.java has all the pokemon names, its types and its weakness.
   |-> pokemon.java has the method to display pokemon's information
   |-> main.java has the template, in which if we give pokemon name as user input, it will display the information from pokemoninfo file.

3. Folder seperateFiles
   |-> This code has been developed from above two set of programs. Here I have used abstraction, polymorphism, inheritence.
   |->I have kept all the pokemon information in seperate files under pokemon folder, which has nearly 30 pokemon characters with its types weakness and special ability.
   |->I have used two methods to display its features and special ability based on user input.
   |-> here I have learnt lot of OOP concepts.
